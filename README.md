# 🎵 Alphabet Music App

An interactive and responsive music web app built using HTML, CSS, and JavaScript.
Each alphabet key plays a unique musical sound with glowing visual effects and keyboard interaction support.

## 🚀 Features

* 🎹 Alphabet based sound system
* ✨ Interactive glowing buttons
* ⌨️ Keyboard support
* 🎵 Real audio effects using MP3 files
* 📱 Responsive design
* 🌌 Modern UI with animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
